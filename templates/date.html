Datetimepicker


El control Datetimepicker permite seleccionar un solo elemento de una lista de fechas u horas.


Ejemplo simple




<input type="datetime-local" id="datetime-local" class="btn btn-light Q">
<input type="date" id="date" class="btn btn-light Q">
<input type="time" id="time" class="btn btn-light Q">

Ejemplo con bordes

Puede agregar la clase .dtp-*con el color contextual para tener bordes. 






<input type="datetime-local" id="datetime-local" class="btn dtp-primary Q">
<input type="datetime-local" id="datetime-local" class="btn dtp-secondary Q">
<input type="datetime-local" id="datetime-local" class="btn dtp-success Q">
<input type="datetime-local" id="datetime-local" class="btn dtp-danger Q">
<input type="datetime-local" id="datetime-local" class="btn dtp-warning Q">
<input type="datetime-local" id="datetime-local" class="btn dtp-info Q">

Ejemplo tipo botón

Puede agregar la clase .btn-*con el color contextual para tener una apariencia de botón. 






<input type="datetime-local" id="datetime-local" class="btn btn-primary Q">
