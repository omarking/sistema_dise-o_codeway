Collapse


Este componente sirve para mostrar contenido muy amplio, pero que de inicio de esta oculto.
Haga clic en los botones de abajo para mostrar y ocultar otro elemento a través de cambios de clase:

  • .collapse oculta el contenido
  • .collapsing se aplica durante las transiciones.
  • .collapse.show muestra contenido

Generalmente, recomendamos usar un botón con el atributo data-bs-target. Aunque no se recomienda desde un punto de vista semántico, también puede utilizar un enlace con el atributo href (y un role = "button").
En ambos casos, se requiere data-bs-toggle = "collapse".

Horizontal

Agregue la clase modificadora .collapse-horizontal para hacer la transición horizontalemente.
Puede especificar en el div principal el espacio apartador para la carta style="min-height: 120px;" y puede especificar el tamaño del cuerpo de la carta style="width: 300px;".

Este es un contenido de marcador de posición para un colapso horizontal. Está oculto de forma predeterminada y se muestra cuando se activa.

</p>

Múltiples cartas

Un <button> o <a> puede mostrar y ocultar varios elementos haciendo referencia a ellos con un selector en su atributo href o data-bs-target.
Varios <button> o <a> pueden mostrar y ocultar un elemento si cada uno hace referencia a él con su atributo href o data-bs-target.
