Filter


Escriba algo en el campo de entrada para buscar en la tabla nombres, apellidos o correos electrónicos:
Tenga en cuenta que comenzamos la búsqueda en tbody, para evitar filtrar los encabezados de la tabla.
Para este ejemplo es necesario importar el cdn de JQuery para hacer el filtro.


Firstname Lastname Email
John Doe john@example.com
Mary Moe mary@mail.com
July Dooley july@greatstuff.com
Anja Ravendale a_r@test.com 
<!--Estructura del filtro-->
<input class="form-control Q" id="myInput" type="text" placeholder="Search..">
<table class="table table-bordered table-striped Q">
    <thead>
        <tr>
            <th>Firstname</th>
            <th>Lastname</th>
            <th>Email</th>
        </tr>
    </thead>
    <tbody id="myTable">
        <tr>
            <td>John</td>
            <td>Doe</td>
            <td>john@example.com</td>
        </tr>
        <tr>
            <td>Mary</td>
            <td>Moe</td>
            <td>mary@mail.com</td>
        </tr>
        <tr>
            <td>July</td>
            <td>Dooley</td>
            <td>july@greatstuff.com</td>
        </tr>
        <tr>
            <td>Anja</td>
            <td>Ravendale</td>
            <td>a_r@test.com</td>
        </tr>
    </tbody>
</table>
<!--Script para filtrar-->
<script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"></script>
<script>
    $(document).ready(function () {
        $("#myInput").on("keyup", function () {
            var value = $(this).val().toLowerCase();
            $("#myTable tr").filter(function () {
                $(this).toggle($(this).text().toLowerCase().indexOf(
                    value) > -1)
            });
        });
    });
</script>
