Listas


Las listas son un componente que ayuda a mostrar información en grupos, por lo general en forma ordenada.


Ejemplo básico


  • An item
  • A second item
  • A third item
  • A fourth item

<ul class="list-group Q">
    <li class="list-group-item">An item</li>
    <li class="list-group-item">A second</li>
    <li class="list-group-item">A third item</li>
    <li class="list-group-item">A fourth item</li>
</ul>

Elementos activos


Agregue .active al .list-group-item para activar el elemento.


  • An item
  • A second item
  • A third item
  • A fourth item

<ul class="list-group Q">
    <li class="list-group-item active">An item</li>
    <li class="list-group-item">A second</li>
    <li class="list-group-item">A third item</li>
    <li class="list-group-item">A fourth item</li>
</ul>

Elementos inactivos


Agregue .disabled al .list-group-item para que los elementos de la lista parezcan deshabilitados.


  • An item
  • A second item
  • A third item
  • A fourth item

<ul class="list-group Q">
    <li class="list-group-item disabled" aria-disabled="true">An item</li>
    <li class="list-group-item">A second</li>
    <li class="list-group-item">A third item</li>
    <li class="list-group-item">A fourth item</li>
</ul>

Enlaces y botones


Puede usar <a> o <button> para crear elementos de listas accionables con estados flotantes,
desactivados y activos agregando .list-group-item-action.


    The current link item A second link item 

<ul class="list-group Q">
    <a href="#" class="list-group-item list-group-item-action active" aria-current="true">
        The current link item
    </a>
    <a href="#" class="list-group-item list-group-item-action">
        A second link item
    </a>
    <button type="button" class="list-group-item list-group-item-action active" aria-current="true">
        The current button
    </button>
    <button type="button" class="list-group-item list-group-item-action">
        A second item
    </button>
</ul>

Flush


Agregue la clase .list-group-flush para eliminar algunos bordes y esquinas.


  • An item
  • A second
  • A third item
  • A fourth item

<ul class="list-group list-group-flush Q">
    <li class="list-group-item">An item</li>
    <li class="list-group-item">A second</li>
    <li class="list-group-item">A third item</li>
    <li class="list-group-item">A fourth item</li>
</ul>

Numerado


Agregue la clase .list-group-nudered (Opcionalmente, use un elemento <ol>)


  1. A list item
  2. A list item
  3. A list item

<ol class="list-group list-group-numbered Q">
    <li class="list-group-item">A list item</li>
    <li class="list-group-item">A list item</li>
    <li class="list-group-item">A list item</li>
</ol>
