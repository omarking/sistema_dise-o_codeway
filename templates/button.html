Buttons


Los botones proporcionan información sobre las acciones que se pueden hacer en pantalla cada uno con su propio propósito semántico, con algunos extras para un mayor control. 




<button type="button" class="btn btn-primary Q">Primary</button>
<button type="button" class="btn btn-secondary Q">Secondary</button>
<button type="button" class="btn btn-success Q">Success</button>
<button type="button" class="btn btn-danger Q">Danger</button>
<button type="button" class="btn btn-warning Q">Warning</button>
<button type="button" class="btn btn-info Q">Info</button>
<button type="button" class="btn btn-light Q">Light</button>
<button type="button" class="btn btn-dark Q">Dark</button>
<button type="button" class="btn btn-link Q">Link</button>

Tamaños

Agregue .btn-lg o .btn-sm para tamaños adicionales. 




<button type="button" class="btn btn-primary btn-lg Q">Large button</button>
<button type="button" class="btn btn-secondary btn-lg Q">Large button</button>
<button type="button" class="btn btn-primary btn-sm Q">Small button</button>
<button type="button" class="btn btn-secondary btn-sm Q">Small button</button>

Estado inactivo

Agregue el atributo booleano disabled a los elementos <buttons>.
Los botones deshabilitados que usan el elemento <a> no son compatibles con el atributo disabled, por lo que debe agregar la clase .disabled para que parezca visualmente deshabilitado.
Deben incluir el atributo aria-disabled = "true" para indicar el estado del elemento a las tecnologías de asistencia.
No deben incluir el atributo href.


Primary link Link 

<button type="button" class="btn btn-lg btn-primary Q" disabled>Primary button</button>
<button type="button" class="btn btn-secondary btn-lg Q" disabled>Button</button>
<a class="btn btn-primary btn-lg disabled Q" role="button" aria-disabled="true">Primary link</a>
<a class="btn btn-secondary btn-lg disabled Q" role="button" aria-disabled="true">Link</a>

Grupo de botones


Ejemplo básico

Envuelva una serie de botones con la clase .btn en .btn-group. 




<div class="btn-group Q" role="group" aria-label="Basic example">
    <button type="button" class="btn btn-primary">Left</button>
    <button type="button" class="btn btn-primary">Middle</button>
    <button type="button" class="btn btn-primary">Right</button>
</div>

Tamaños

En lugar de aplicar clases de tamaño de botón a cada botón de un grupo, simplemente agregue .btn-group-* (lg o sm). 




<div class="btn-group btn-group-lg" role="group" aria-label="Basic example">...</div>
<div class="btn-group btn-group" role="group" aria-label="Basic example">...</div>
<div class="btn-group btn-group-sm" role="group" aria-label="Basic example">...</div>
