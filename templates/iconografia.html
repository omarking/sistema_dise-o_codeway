Iconografia


La librería a utilizar para integrar iconos dentro de nuestra web es iconografía de Ionic, para ello deves refenciar la siguiente clase: Ionicons 

<head>
    <script type="module" src="https://unpkg.com/ionicons@5.5.2/dist/ionicons/ionicons.esm.js"></script>
    <script nomodule src="https://unpkg.com/ionicons@5.5.2/dist/ionicons/ionicons.js"></script>
</head>

Principales Usos

  • Navegación principal
  • Input text
  • Botones
  • Dropdowns
  • Apoyando algunos títulos
  • Acompañando algunos textos explicativos


Uso básico

Para usar un ícono integrado del paquete Ionicons, complete el atributo de nombre en el componente ion-icon: 


<ion-icon name="heart"></ion-icon>

Variantes

Cada ícono de aplicación en Ionicons tiene una variante filled, outline y sharp. Estas diferentes variantes se proporcionan para que su aplicación se sienta nativa de una variedad de plataformas. La variante filled usa el nombre predeterminado sin sufijo. 


<ion-icon name="heart"></ion-icon> <!--filled-->
<ion-icon name="heart-outline"></ion-icon> <!--outline-->
<ion-icon name="heart-sharp"></ion-icon> <!--sharp-->

Tamaño

Para especificar el tamaño del icono, puede utilizar el atributo de tamaño para nuestros tamaños de fuente predefinidos. 

<ion-icon name="heart" size="small"></ion-icon>
<ion-icon name="heart" size="large"></ion-icon>

Tambien puede establecer un tamaño específico aplicando la propiedad CSS de tamaño de fuente en el componente ion-icon.
Se recomienda utilizar tamaños de píxeles que sean múltiplos de 8 (8, 16, 32, 64, etc.) 

ion-icon {
  font-size: 64px;
}

Color

Especifique el color del icono aplicando la propiedad CSS de color en el componente ion-icon. 

ion-icon {
    color: blue;
}

Iconos

Puede acceder al sitio oficial de Ionic y seleccionar el icono que necesite y en automatico obtengra el codigo para implementarlo.
